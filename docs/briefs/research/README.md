# Research Briefs

This directory contains topic dossiers aimed at Ishoo's next stage of product and architecture work.

Each dossier follows the same pattern:

1. define six concrete research questions for one topic
2. answer each question with a short report grounded in current sources
3. compile those findings into a topic-level synthesis for Ishoo

Current research set:

- [Agent-Native Issue Operations](./agent-native-issue-operations.md)
- [Local-First and Git-Backed Architecture](./local-first-git-backed-architecture.md)
- [Desktop Interaction, Accessibility, and Visualization](./desktop-interaction-accessibility-and-visualization.md)

These are not raw transcript dumps. They are distilled toward Ishoo's actual surface:

- markdown issue records
- CLI-first and AI-agent workflows
- local-first persistence
- Git-based change management
- feed/board interaction quality
- dependency and hotspot visualization
